# MTG Commander Deck Builder

A Squad sample that scrapes **EDHREC** card recommendations using Playwright, then hands them to a four-agent AI squad to build a complete 100-card Commander deck. Supports follow-up modifications in a conversation loop.

## How It Works

1. You describe a deck concept or name a specific Commander
2. A Chromium browser launches and navigates to EDHREC
3. The app scrapes recommended cards (names, categories, inclusion rates)
4. Browser closes, then four AI agents collaborate to build your deck:
   - **Card Scout** — finds synergistic cards and staples for your commander/theme
   - **Deck Architect** — builds the mana curve, land base, and overall structure
   - **Synergy Analyst** — identifies combos, win conditions, and card interactions
   - **Budget Advisor** — suggests budget alternatives and flags expensive cards
5. The complete deck is saved as JSON and a formatted text file
6. You can keep modifying the deck with follow-up commands ("add more card draw", "swap expensive cards for budget alternatives")

## Prerequisites

- Node.js ≥ 20
- GitHub Copilot CLI installed and authenticated
- Playwright (installed automatically via `postinstall`)

## Setup

```bash
npm install
```

Playwright's Chromium browser is installed automatically. If needed, run manually:

```bash
npx playwright install chromium
```

## Usage

```bash
npm start
```

## Deck File Format

Decks are saved in the `decks/` directory in two formats:

### JSON (`decks/{name}.json`)

Contains structured data with metadata and card objects:

```json
{
  "metadata": {
    "commander": "Atraxa, Praetors' Voice",
    "theme": "Superfriends",
    "colors": ["W", "U", "B", "G"],
    "createdAt": "2026-03-15T10:30:00.000Z",
    "lastModified": "2026-03-15T10:35:00.000Z",
    "totalCards": 100
  },
  "cards": [
    { "name": "Atraxa, Praetors' Voice", "quantity": 1, "category": "commander" },
    { "name": "Deepglow Skate", "quantity": 1, "category": "creature" }
  ]
}
```

### Text (`decks/{name}.txt`)

Human-readable deck list:

```
// MTG Commander Deck: Atraxa, Praetors' Voice
// Theme: Superfriends
// Colors: W/U/B/G
// Total Cards: 100
// Generated by Squad MTG Deck Builder

=== COMMANDER (1) ===
1x Atraxa, Praetors' Voice

=== CREATURES (25) ===
1x Deepglow Skate
1x Evolution Sage
...

=== LANDS (36) ===
1x Command Tower
1x Breeding Pool
...
```

## Notes

- **Read-only scraping** — this demo reads EDHREC but doesn't modify anything or require an account
- **Extensible** — add TCGPlayer API tools, collection tracking, or export to deck-building sites
- **Privacy** — scraped card data is sent to the AI model for deck building but is not stored beyond your local `decks/` directory
- **EDHREC DOM** — EDHREC's interface changes occasionally; the scraper uses multiple fallback strategies
- **Session persistence** — browser profile is saved in `.edhrec-session/` to preserve cookie consent across runs
