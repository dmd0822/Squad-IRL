# Composting Guide Generator - Specification

**Audience:** 🌱 Environmental  
**Goal:** Create personalized composting setup guides based on space, goals, and experience level

## Requirements

1. Assess available space (backyard, balcony, indoor, etc.)
2. Recommend composting method (bin, tumbler, vermicompost, etc.)
3. List required materials and equipment with costs
4. Explain what can/cannot be composted (greens, browns, no-nos)
5. Provide setup instructions step-by-step
6. Include maintenance routine (turning, moisture, troubleshooting)
7. Address common problems (smell, pests, slow decomposition)
8. Estimate timeframe to finished compost

## Output Format

Personalized composting guide with method selection, setup, maintenance, and troubleshooting

## Quality Criteria

- **Method Appropriateness (30)**: Recommended method matches space and lifestyle constraints
- **Educational Value (25)**: Clearly explains composting science; empowers user to problem-solve
- **Completeness (20)**: Covers setup, materials, what to compost, maintenance, troubleshooting
- **Practical Detail (15)**: Specific instructions, not vague; includes costs and timeline
- **Troubleshooting (10)**: Addresses common problems with solutions


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
