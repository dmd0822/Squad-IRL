# Estate Planning Checklist

> Comprehensive estate planning task list with priority and timeline

## Who This Is For

👵 Seniors/Adults

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| EstatePlanner | Writer | Domain expert for estate planning checklist |
| Organizer | Grader | Domain expert for estate planning checklist |
| DocumentTracker | Updater | Domain expert for estate planning checklist |

## How It Works

1. **Write** — EstatePlanner reads `spec.md` and produces the first draft
2. **Grade** — Organizer evaluates the output (1-100 score)
3. **Update** — If score < 90%, DocumentTracker improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
