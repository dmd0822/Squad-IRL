# Competitor Analysis Digest

> Weekly competitor monitoring with strategic insights

## Who This Is For

🏪 Small Business

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| SpyBot | Writer | Domain expert for competitor analysis digest |
| StrategyAnalyst | Grader | Domain expert for competitor analysis digest |
| OpportunitySpotter | Updater | Domain expert for competitor analysis digest |

## How It Works

1. **Write** — SpyBot reads `spec.md` and produces the first draft
2. **Grade** — StrategyAnalyst evaluates the output (1-100 score)
3. **Update** — If score < 90%, OpportunitySpotter improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
