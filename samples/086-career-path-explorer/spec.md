# Career Path Explorer - Specification

**Audience:** 🎓 Professionals  
**Goal:** Map realistic career paths with required skills, timelines, and transition strategies

## Requirements

1. Define target career goal and current position
2. Identify skill gaps and learning requirements
3. Provide realistic timeline with milestones
4. Suggest specific certifications, courses, or degrees needed
5. Map potential intermediate roles as stepping stones
6. Include salary progression and market demand insights
7. Address common challenges and how to overcome them
8. Suggest networking strategies and industry connections to build

## Output Format

Visual career path map with skill requirements, timelines, and actionable next steps

## Quality Criteria

- **Realism & Accuracy (30)**: Timeline and requirements reflect actual industry standards; no false promises
- **Skill Gap Analysis (25)**: Clear identification of what needs to be learned; specific resources provided
- **Pathway Clarity (20)**: Logical progression with intermediate milestones; not just start-to-finish
- **Market Intelligence (15)**: Includes salary data, demand outlook, and industry trends
- **Actionability (10)**: Specific next steps with resources, not vague advice


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
