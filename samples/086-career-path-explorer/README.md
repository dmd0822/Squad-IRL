# Career Path Explorer

> Map potential career paths with skills needed and realistic timelines

## Who This Is For

🎓 Students/Professionals

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| CareerCoach | Writer | Domain expert for career path explorer |
| ResearchBot | Grader | Domain expert for career path explorer |
| RealityChecker | Updater | Domain expert for career path explorer |

## How It Works

1. **Write** — CareerCoach reads `spec.md` and produces the first draft
2. **Grade** — ResearchBot evaluates the output (1-100 score)
3. **Update** — If score < 90%, RealityChecker improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
