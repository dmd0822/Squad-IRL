# Zero Waste Swap Finder

> Identify eco-friendly product swaps for everyday items

## Who This Is For

🌱 Environmental

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| EcoExpert | Writer | Domain expert for zero waste swap finder |
| ProductResearcher | Grader | Domain expert for zero waste swap finder |
| ImpactCalculator | Updater | Domain expert for zero waste swap finder |

## How It Works

1. **Write** — EcoExpert reads `spec.md` and produces the first draft
2. **Grade** — ProductResearcher evaluates the output (1-100 score)
3. **Update** — If score < 90%, ImpactCalculator improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
