# Newsletter Writer

> Transform business updates into engaging email newsletters

## Who This Is For

🏪 Small Business

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Reporter | Writer | Domain expert for newsletter writer |
| Storyteller | Grader | Domain expert for newsletter writer |
| CTAExpert | Updater | Domain expert for newsletter writer |

## How It Works

1. **Write** — Reporter reads `spec.md` and produces the first draft
2. **Grade** — Storyteller evaluates the output (1-100 score)
3. **Update** — If score < 90%, CTAExpert improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
