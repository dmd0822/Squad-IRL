# Social Media Content Calendar

> Generate 30 days of on-brand social posts with captions and hashtags

## Who This Is For

🏪 Small Business

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Creator | Writer | Domain expert for social media content calendar |
| BrandVoice | Grader | Domain expert for social media content calendar |
| Strategist | Updater | Domain expert for social media content calendar |
| Scheduler | Specialist | Domain expert for social media content calendar |

## How It Works

1. **Write** — Creator reads `spec.md` and produces the first draft
2. **Grade** — BrandVoice evaluates the output (1-100 score)
3. **Update** — If score < 90%, Strategist improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
