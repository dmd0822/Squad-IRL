# Side Hustle Validator

> Evaluate side business ideas for market fit, competition, and profitability

## Who This Is For

Everyone

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| MarketResearcher | Writer | Domain expert for side hustle validator |
| Realist | Grader | Domain expert for side hustle validator |
| NumbersCruncher | Updater | Domain expert for side hustle validator |

## How It Works

1. **Write** — MarketResearcher reads `spec.md` and produces the first draft
2. **Grade** — Realist evaluates the output (1-100 score)
3. **Update** — If score < 90%, NumbersCruncher improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
