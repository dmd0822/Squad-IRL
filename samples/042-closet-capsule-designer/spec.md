# Closet Capsule Designer Specification

## Goal
Design a minimalist capsule wardrobe from the user's existing clothing, maximizing outfit combinations while minimizing clutter.

## Requirements

1. **Inventory Analysis** - Analyze the user's current wardrobe (items, colors, condition, fit)
2. **Style Profile** - Identify user's lifestyle needs (work, casual, formal) and personal style preferences
3. **Color Palette** - Create a cohesive 4-6 color palette that allows maximum mixing
4. **Core Pieces** - Identify 15-30 essential items that form the capsule foundation
5. **Mix-Match Formula** - Show how core pieces combine to create 30+ outfits
6. **Remove/Store List** - Identify items to remove (donate/sell) or store seasonally
7. **Gap Analysis** - Identify any missing pieces needed to complete the capsule
8. **Outfit Formulas** - Provide 5-7 versatile outfit formulas (e.g., "blazer + tee + jeans")
9. **Seasonal Adaptability** - Show how capsule adapts across seasons with minimal additions
10. **Visual Organization** - Suggest closet organization strategy for easy outfit assembly

## Output Format

```markdown
# Your Capsule Wardrobe Plan

## Your Style Profile
[Lifestyle needs, style preferences, body type considerations]

## Color Palette
[4-6 core colors with hex codes, why they work together]

## The Capsule (15-30 pieces)
### Tops (5-8 items)
### Bottoms (4-6 items)
### Layers (3-5 items)
### Shoes (3-4 pairs)
### Accessories (2-4 items)

## 30+ Outfit Combinations
[Visual grid or list showing how pieces combine]

## Action Plan
- **Remove/Donate**: [Items that don't fit the capsule]
- **Store Seasonally**: [Off-season items]
- **Shopping List**: [2-5 key gaps to fill]

## Outfit Formulas
[7 versatile formulas for different occasions]
```

## Quality Criteria

- **Versatility**: Each item should pair with at least 5 other pieces
- **Realism**: Recommendations respect user's lifestyle and body type
- **Color Harmony**: All items coordinate within the palette
- **Practicality**: Outfits are appropriate for user's actual life
- **Minimalism**: Focus on fewer, better pieces rather than more options
