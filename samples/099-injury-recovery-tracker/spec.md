# Injury Recovery Tracker - Specification

**Audience:** 🏃 Health  
**Goal:** Track injury recovery progress and provide realistic return-to-activity timelines with milestones

## Requirements

1. Define injury type and severity baseline
2. Establish recovery milestones (phases: acute, rehab, return-to-sport)
3. Track pain levels, range of motion, and functional tests
4. Provide rehab exercise progression based on progress
5. Set realistic timeline with conservative and optimistic scenarios
6. Identify setback warning signs and when to rest
7. Suggest modifications for maintaining fitness during recovery
8. Provide criteria for safe return to full activity

## Output Format

Recovery tracker with milestones, progress metrics, timeline, and return-to-activity criteria

## Quality Criteria

- **Clinical Accuracy (30)**: Recovery phases and timelines are realistic for injury type; safe protocols
- **Milestone Clarity (25)**: Clear, measurable milestones tied to recovery phases
- **Progress Tracking (20)**: Effective system for monitoring pain, ROM, strength, function
- **Safety Focus (15)**: Identifies setback warning signs; emphasizes gradual progression
- **Practicality (10)**: Realistic for self-tracking; includes fitness maintenance options


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
