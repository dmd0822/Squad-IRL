# 100 Ways to Use Squad — Master List

> Real-world automation for real people. No coding required.

## How Each Sample Works

Every sample follows the same pattern:
1. **Writer Agent** — reads a spec and produces the first draft
2. **Grader Agent** — evaluates output quality against the spec (1-100%)
3. **Updater Agent** — improves the output if grade < 90%
4. Repeat until quality threshold is met

## What Makes These Different

Unlike traditional automation tools, Squad SDK samples show **collaborative AI teams** working together with:
- **Self-improving loops** — quality gates ensure output meets standards
- **Domain expertise** — each agent brings specialized knowledge
- **Human-in-the-loop** — you guide the team, they do the work
- **Reusable patterns** — adapt any sample to your specific needs

## Next Steps

Each sample will have its own directory in `samples/` with:
- `charter.md` — Agent team configuration
- `README.md` — Usage instructions and examples
- Sample input/output files
