# Dating Profile Optimizer

> Polish dating profiles to be authentic yet compelling

## Who This Is For

Everyone

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Matchmaker | Writer | Domain expert for dating profile optimizer |
| AuthenticityGuard | Grader | Domain expert for dating profile optimizer |
| PhotoAnalyst | Updater | Domain expert for dating profile optimizer |

## How It Works

1. **Write** — Matchmaker reads `spec.md` and produces the first draft
2. **Grade** — AuthenticityGuard evaluates the output (1-100 score)
3. **Update** — If score < 90%, PhotoAnalyst improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```
