# Portfolio Website Copy Writer - Specification

**Audience:** 🎨 Freelancers  
**Goal:** Generate compelling, authentic portfolio website copy that showcases skills and personality

## Requirements

1. Extract key skills, experiences, and personality traits from raw bio/CV
2. Write engaging 'About Me' section that balances professionalism with personality
3. Create project descriptions that highlight impact and outcomes
4. Include SEO-optimized keywords naturally without sounding forced
5. Craft compelling CTAs (call-to-action) for contact/hire sections
6. Maintain consistent voice and tone throughout all sections
7. Ensure copy is scannable with clear hierarchy and formatting

## Output Format

Complete website copy sections (Hero, About, Projects, Services, Contact) in markdown

## Quality Criteria

- **Personality & Authenticity (30)**: Voice feels genuine, not templated; unique personality shines through
- **Clarity & Impact (25)**: Value proposition is immediately clear; accomplishments are specific and impressive
- **SEO & Discoverability (20)**: Keywords integrated naturally; readable by both humans and search engines
- **Structure & Scannability (15)**: Clear hierarchy, short paragraphs, effective use of formatting
- **Call-to-Action Effectiveness (10)**: CTAs are compelling and appropriately placed


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
