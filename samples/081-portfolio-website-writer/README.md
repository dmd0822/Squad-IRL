# Portfolio Website Writer

> Write compelling copy for portfolio websites that convert visitors

## Who This Is For

🎨 Artists/Freelancers

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Copywriter | Writer | Domain expert for portfolio website writer |
| SEOExpert | Grader | Domain expert for portfolio website writer |
| PersonalityCapture | Updater | Domain expert for portfolio website writer |

## How It Works

1. **Write** — Copywriter reads `spec.md` and produces the first draft
2. **Grade** — SEOExpert evaluates the output (1-100 score)
3. **Update** — If score < 90%, PersonalityCapture improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
