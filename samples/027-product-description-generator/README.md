# Product Description Generator

> Write SEO-optimized product descriptions that convert browsers to buyers

## Who This Is For

🏪 Small Business

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Copywriter | Writer | Domain expert for product description generator |
| SEOExpert | Grader | Domain expert for product description generator |
| ConversionOptimizer | Updater | Domain expert for product description generator |

## How It Works

1. **Write** — Copywriter reads `spec.md` and produces the first draft
2. **Grade** — SEOExpert evaluates the output (1-100 score)
3. **Update** — If score < 90%, ConversionOptimizer improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
