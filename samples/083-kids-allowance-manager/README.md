# Kids Allowance Manager

> Create allowance systems that teach financial responsibility

## Who This Is For

👩‍👧‍👦 Parents

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| FinancialEducator | Writer | Domain expert for kids allowance manager |
| FairnessJudge | Grader | Domain expert for kids allowance manager |
| GoalSetter | Updater | Domain expert for kids allowance manager |

## How It Works

1. **Write** — FinancialEducator reads `spec.md` and produces the first draft
2. **Grade** — FairnessJudge evaluates the output (1-100 score)
3. **Update** — If score < 90%, GoalSetter improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
