# Kids' Allowance System Designer - Specification

**Audience:** 👩‍👧‍👦 Parents  
**Goal:** Design age-appropriate allowance systems that teach financial responsibility and goal-setting

## Requirements

1. Create age-appropriate earning/saving/spending framework
2. Balance chores-based vs. unconditional allowance models
3. Include savings goals and tracking mechanisms
4. Incorporate lessons on giving/charity (e.g., 10% rule)
5. Define clear expectations and consequences
6. Suggest tools or apps for tracking (or DIY methods)
7. Provide conversation starters for money discussions
8. Scale system as child matures (age progression)

## Output Format

Complete allowance system guide with tracking templates and discussion prompts

## Quality Criteria

- **Age Appropriateness (30)**: System matches child's developmental stage; complexity is right-sized
- **Educational Value (25)**: Teaches earning, saving, spending, giving; includes goal-setting
- **Fairness & Clarity (20)**: Rules are clear, consistent, and perceived as fair by kids
- **Practical Implementation (15)**: Includes tracking tools and is realistic for busy families
- **Growth & Flexibility (10)**: System can evolve as child matures; built-in progression


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
