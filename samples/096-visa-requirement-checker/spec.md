# Visa Requirement Checker - Specification

**Audience:** ✈️ Travelers  
**Goal:** Provide accurate visa requirements and complete application checklists for international travel

## Requirements

1. Determine visa requirement based on passport and destination
2. Specify visa type needed (tourist, business, transit, etc.)
3. List all required documents (passport, photos, financial proof, etc.)
4. Provide application process steps and timeline
5. Include fees and payment methods
6. Note processing time and recommend when to apply
7. Highlight common rejection reasons and how to avoid them
8. Provide embassy/consulate contact info and appointment booking details

## Output Format

Visa guide with requirements, document checklist, timeline, and application instructions

## Quality Criteria

- **Accuracy (30)**: Visa requirements are current and correct for passport/destination combo
- **Document Completeness (25)**: All required docs listed with specs (photo size, bank statement recency, etc.)
- **Process Clarity (20)**: Step-by-step instructions are clear and complete
- **Timeline Guidance (15)**: Processing time and application timing recommendations are realistic
- **Practical Support (10)**: Includes fees, contacts, common pitfalls, and how to avoid rejection


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
