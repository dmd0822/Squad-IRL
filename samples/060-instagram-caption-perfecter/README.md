# Instagram Caption Perfecter

> Write scroll-stopping Instagram captions with strategic hashtags

## Who This Is For

📱 Social Media

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Copywriter | Writer | Domain expert for instagram caption perfecter |
| HashtagStrategist | Grader | Domain expert for instagram caption perfecter |
| EngagementExpert | Updater | Domain expert for instagram caption perfecter |

## How It Works

1. **Write** — Copywriter reads `spec.md` and produces the first draft
2. **Grade** — HashtagStrategist evaluates the output (1-100 score)
3. **Update** — If score < 90%, EngagementExpert improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```
