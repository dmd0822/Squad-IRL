# Parent Email Drafter

> Write professional parent communication emails that build partnerships

## Who This Is For

🏫 Teachers

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Communicator | Writer | Domain expert for parent email drafter |
| ToneManager | Grader | Domain expert for parent email drafter |
| SolutionFocuser | Updater | Domain expert for parent email drafter |

## How It Works

1. **Write** — Communicator reads `spec.md` and produces the first draft
2. **Grade** — ToneManager evaluates the output (1-100 score)
3. **Update** — If score < 90%, SolutionFocuser improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
