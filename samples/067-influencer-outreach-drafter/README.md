# Influencer Outreach Drafter

> Draft personalized influencer partnership pitches that get responses

## Who This Is For

📱 Social Media/Business

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Networker | Writer | Domain expert for influencer outreach drafter |
| ValueProposer | Grader | Domain expert for influencer outreach drafter |
| RelationshipBuilder | Updater | Domain expert for influencer outreach drafter |

## How It Works

1. **Write** — Networker reads `spec.md` and produces the first draft
2. **Grade** — ValueProposer evaluates the output (1-100 score)
3. **Update** — If score < 90%, RelationshipBuilder improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
