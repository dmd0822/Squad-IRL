# Moving Checklist Generator

> Comprehensive moving checklist with timeline and vendor coordination

## Who This Is For

🏠 Everyone

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Organizer | Writer | Domain expert for moving checklist generator |
| TimelineBoss | Grader | Domain expert for moving checklist generator |
| StressReducer | Updater | Domain expert for moving checklist generator |

## How It Works

1. **Write** — Organizer reads `spec.md` and produces the first draft
2. **Grade** — TimelineBoss evaluates the output (1-100 score)
3. **Update** — If score < 90%, StressReducer improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
