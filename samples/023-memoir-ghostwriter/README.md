# Memoir Ghostwriter

> Interview-based memoir writing that captures personal stories

## Who This Is For

👵 Seniors

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Interviewer | Writer | Domain expert for memoir ghostwriter |
| Storyteller | Grader | Domain expert for memoir ghostwriter |
| FactChecker | Updater | Domain expert for memoir ghostwriter |
| Polisher | Specialist | Domain expert for memoir ghostwriter |

## How It Works

1. **Write** — Interviewer reads `spec.md` and produces the first draft
2. **Grade** — Storyteller evaluates the output (1-100 score)
3. **Update** — If score < 90%, FactChecker improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
