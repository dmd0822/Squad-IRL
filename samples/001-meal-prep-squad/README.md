# Meal Prep Squad

> Weekly meal planning with nutrition goals, budget constraints, and picky eaters

## Who This Is For

👩‍👧‍👦 Parents/Families

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Chef | Writer | Domain expert for meal prep squad |
| Nutritionist | Grader | Domain expert for meal prep squad |
| Shopper | Updater | Domain expert for meal prep squad |
| Budgeteer | Specialist | Domain expert for meal prep squad |

## How It Works

1. **Write** — Chef reads `spec.md` and produces the first draft
2. **Grade** — Nutritionist evaluates the output (1-100 score)
3. **Update** — If score < 90%, Shopper improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
