# Budget Accountability Coach

> Weekly spending review with tough love and actionable advice

## Who This Is For

💰 Finance/Everyone

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Coach | Writer | Domain expert for budget accountability coach |
| Realist | Grader | Domain expert for budget accountability coach |
| Motivator | Updater | Domain expert for budget accountability coach |

## How It Works

1. **Write** — Coach reads `spec.md` and produces the first draft
2. **Grade** — Realist evaluates the output (1-100 score)
3. **Update** — If score < 90%, Motivator improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

Results are saved in the `output/` directory.
