# Fundraiser Campaign Writer

> Write compelling fundraiser stories and updates that drive donations

## Who This Is For

🏘️ Community

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Storyteller | Writer | Domain expert for fundraiser campaign writer |
| EmotionExpert | Grader | Domain expert for fundraiser campaign writer |
| DonorPsychologist | Updater | Domain expert for fundraiser campaign writer |

## How It Works

1. **Write** — Storyteller reads `spec.md` and produces the first draft
2. **Grade** — EmotionExpert evaluates the output (1-100 score)
3. **Update** — If score < 90%, DonorPsychologist improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
