# Specification: Tournament Bracket Analyzer

## Goal
Analyze tournament matchups and predict outcomes based on player/team statistics, historical performance, head-to-head records, and contextual factors with probabilistic confidence levels.

## Requirements
1. **Matchup Analysis**: Deep dive into each bracket pairing with key factors
2. **Statistical Comparison**: Side-by-side stats for relevant performance metrics
3. **Head-to-Head History**: Past encounters between competitors with outcomes
4. **Form Assessment**: Recent performance trends (hot streaks, slumps)
5. **Contextual Factors**: Injuries, home advantage, rest days, weather, stakes
6. **Probability Estimates**: Win percentage for each competitor with confidence level
7. **Upset Potential**: Identify likely upsets with reasoning
8. **Bracket Progression**: Predict advancement through multiple rounds
9. **Key Matchups**: Highlight pivotal games that affect bracket outcomes
10. **Narrative Elements**: Storylines and rivalries that add context

## Input Format
- Tournament bracket structure
- Competitor names and seedings
- Historical statistics and records
- Recent performance data
- Contextual information (injuries, venue, etc.)
- Sport/game-specific metrics

## Output Format
Analysis report with:
- Round-by-round matchup breakdowns
- Win probability for each game
- Statistical comparison tables
- Upset alerts with reasoning
- Predicted bracket progression
- Championship prediction with path
- Key games to watch
- Confidence ratings per prediction

## Quality Criteria
- Data-driven predictions with clear methodology
- Balanced statistical and contextual analysis
- Transparent about uncertainty
- Identifies genuine upset opportunities
- Engaging narrative alongside numbers
