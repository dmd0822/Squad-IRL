# Medication Reminder System - Specification

**Audience:** 👵 Seniors  
**Goal:** Create comprehensive medication schedules with reminders, interactions checking, and refill tracking

## Requirements

1. Organize all medications with dosages and timing
2. Create daily schedule with specific take-times
3. Check for potential drug interactions or conflicts
4. Note special instructions (with food, on empty stomach, etc.)
5. Set up reminder system (pills, alarms, apps, caregiver help)
6. Track refill dates and prescription renewals
7. Include prescriber contact info for each medication
8. Provide missed-dose protocols

## Output Format

Medication schedule with daily timeline, interaction notes, reminders setup, and refill tracker

## Quality Criteria

- **Organization & Clarity (30)**: Schedule is easy to follow; medications are clearly organized by time
- **Safety (25)**: Interactions flagged; special instructions noted; missed-dose guidance provided
- **Reminder System (20)**: Practical reminder approach suited to patient's abilities and lifestyle
- **Refill Management (15)**: Tracks refills and renewal dates; prevents running out
- **Accessibility (10)**: Senior-friendly format; large text, simple language, visual aids


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
