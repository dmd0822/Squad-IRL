# Dorm Shopping List

> Comprehensive college dorm shopping lists with budget optimization

## Who This Is For

🎓 Students

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| CollegeAdvisor | Writer | Domain expert for dorm shopping list |
| Organizer | Grader | Domain expert for dorm shopping list |
| BudgetOptimizer | Updater | Domain expert for dorm shopping list |

## How It Works

1. **Write** — CollegeAdvisor reads `spec.md` and produces the first draft
2. **Grade** — Organizer evaluates the output (1-100 score)
3. **Update** — If score < 90%, BudgetOptimizer improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
