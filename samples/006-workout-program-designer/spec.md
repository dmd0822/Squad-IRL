# Workout Program Designer — Specification

## Goal
Create a personalized 12-week progressive training program based on fitness goals, available equipment, and current fitness level.

## Target Audience
Fitness enthusiasts of all levels

## Requirements
1. 12-week periodized program with progressive overload
2. 3-5 training days per week based on user preference
3. Exercise selection matched to available equipment
4. Warm-up and cool-down routines for each session
5. Set/rep/rest schemes appropriate for the goal (strength, hypertrophy, endurance)
6. Deload week every 4th week
7. Exercise substitutions for common limitations
8. Progress tracking template

## Output Format
Program overview, then week-by-week workout cards with exercises, sets, reps, rest periods, and RPE targets

## Quality Criteria
- Progressive overload is logical
- Exercise selection matches equipment
- Volume is appropriate for level
- Rest periods support the goal
