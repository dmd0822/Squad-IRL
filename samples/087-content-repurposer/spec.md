# Content Repurposer - Specification

**Audience:** 📱 Social Media  
**Goal:** Transform long-form content into optimized posts for different social media platforms

## Requirements

1. Adapt content for platform-specific formats (Twitter/X, LinkedIn, Instagram, TikTok, etc.)
2. Maintain core message while adjusting tone for each platform's culture
3. Optimize length, hashtags, and formatting per platform
4. Identify key quotes or moments for standalone posts
5. Create platform-appropriate hooks/openings
6. Suggest visual or media elements for each platform
7. Provide posting schedule recommendations
8. Include engagement prompts (questions, CTAs) appropriate to each platform

## Output Format

Platform-specific post variations with hashtags, timing, and visual suggestions

## Quality Criteria

- **Platform Optimization (30)**: Each version feels native to its platform; format and tone are right
- **Message Consistency (25)**: Core value maintained across platforms despite different expressions
- **Engagement Potential (20)**: Hooks are strong; CTAs are platform-appropriate
- **Completeness (15)**: Covers major platforms; includes hashtags, visuals, timing
- **Strategic Coherence (10)**: Posts work as coordinated campaign, not random adaptations


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
