# Screen Time Negotiator — Specification

## Goal
Create a comprehensive, actionable screen time negotiator output tailored to the user's specific situation and needs.

## Target Audience
Parents of kids 6-16

## Requirements
1. Gather and analyze all relevant input information
2. Apply domain expertise in family agreements, age-appropriate limits, digital wellness
3. Produce structured, detailed output organized by priority
4. Include specific recommendations with rationale
5. Provide actionable next steps with timelines
6. Consider budget/resource constraints
7. Address common pitfalls and how to avoid them
8. Include a quick-reference summary for easy scanning

## Output Format
Structured document with executive summary, detailed analysis sections, recommendations table, and action checklist.

## Quality Criteria
- Every input data point addressed
- Recommendations are specific (not "consider your options")
- Numbers, dates, and names used where applicable
- Actionable by someone with no domain expertise
- Formatted for easy scanning and reference
