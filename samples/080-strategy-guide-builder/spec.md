# Specification: Strategy Guide Builder for Games

## Goal
Create comprehensive, well-organized game strategy guides that help players master mechanics, optimize builds/strategies, and progress efficiently through content.

## Requirements
1. **Beginner to Advanced**: Layer information for multiple skill levels
2. **Core Mechanics**: Explain fundamental systems thoroughly
3. **Optimal Strategies**: Best practices for progression, combat, economy, etc.
4. **Build Guides**: Character/class builds with rationale
5. **Progression Path**: Efficient route through game content
6. **Tips & Tricks**: Lesser-known mechanics or exploits
7. **Common Mistakes**: Pitfalls to avoid
8. **Meta Analysis**: Current competitive meta (for multiplayer)
9. **Visual Organization**: Tables, tier lists, stat comparisons
10. **Searchable Structure**: Clear sections and table of contents

## Input Format
- Game title and genre
- Specific aspect to guide (full game, character, mode, etc.)
- Target audience (beginners, advanced, speedrunners)
- Current game version/patch
- Competitive meta information (if applicable)
- Known strategies or builds

## Output Format
Strategy guide with:
- Table of contents
- Introduction and guide scope
- Core mechanics explanation
- Beginner section (getting started, early game)
- Intermediate strategies
- Advanced tactics and optimizations
- Build recommendations with analysis
- Progression roadmap
- Tips and tricks section
- Common mistakes to avoid
- Glossary of terms
- Credits and version info

## Quality Criteria
- Accurate and tested information
- Well-organized and navigable
- Appropriate depth for target audience
- Actionable strategies, not just descriptions
- Up-to-date with current game version
