# DIY Project Sequencer - Specification

**Audience:** 🏠 Homeowners  
**Goal:** Break down DIY home projects into clear sequential steps with tools, materials, and time estimates

## Requirements

1. List all required tools and materials upfront
2. Sequence steps in logical order with dependencies
3. Provide realistic time estimates for each step
4. Include safety warnings and protective equipment needs
5. Specify skill level required (beginner, intermediate, advanced)
6. Identify decision points or customization options
7. Include troubleshooting tips for common issues
8. Suggest when to call a professional instead

## Output Format

Step-by-step project guide with materials list, timeline, and safety notes

## Quality Criteria

- **Sequence Logic (30)**: Steps are in correct order; dependencies are clear; nothing missing
- **Completeness (25)**: All tools, materials, and supplies listed; time estimates included
- **Safety Awareness (20)**: Appropriate warnings; protective equipment specified; hazards identified
- **Clarity (15)**: Steps are specific and actionable; not too vague or too overwhelming
- **Practical Wisdom (10)**: Includes pro tips, troubleshooting, and when to call a pro


## Example Context

The writer agent will receive contextual information (e.g., user profile, preferences, data) and must produce output that fully satisfies all requirements. The grader will evaluate strictly against these criteria.
