# College App Coach

> Guide students through college application process with timeline and essay help

## Who This Is For

🎓 Students

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Advisor | Writer | Domain expert for college app coach |
| EssayCoach | Grader | Domain expert for college app coach |
| Cheerleader | Updater | Domain expert for college app coach |
| Deadline Sheriff | Specialist | Domain expert for college app coach |

## How It Works

1. **Write** — Advisor reads `spec.md` and produces the first draft
2. **Grade** — EssayCoach evaluates the output (1-100 score)
3. **Update** — If score < 90%, Cheerleader improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
