# Petition Drafter

> Write compelling petitions for local change initiatives

## Who This Is For

🏘️ Community

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Organizer | Writer | Domain expert for petition drafter |
| PersuasionExpert | Grader | Domain expert for petition drafter |
| CallToActionWriter | Updater | Domain expert for petition drafter |

## How It Works

1. **Write** — Organizer reads `spec.md` and produces the first draft
2. **Grade** — PersuasionExpert evaluates the output (1-100 score)
3. **Update** — If score < 90%, CallToActionWriter improves the output
4. **Repeat** — Loop continues until quality threshold is met

## Running This Sample

```bash
npm install
npm start
```

## Output

Results are saved in the `output/` directory.
