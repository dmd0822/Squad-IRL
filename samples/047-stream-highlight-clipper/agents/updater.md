# Stream Highlight Clipper Update Guidance

## Update Strategy

When feedback indicates issues:

1. **Refine Selection** - Replace low-energy clips with better moments; ensure variety
2. **Improve Titles** - Make titles more engaging while keeping them accurate
3. **Enhance Platform Fit** - Add specific length/format recommendations for each platform
4. **Add Context** - Explain what happened more clearly for non-viewers
5. **Better Thumbnails** - Choose more expressive freeze-frames

## Focus Areas

- **If scored low on Selection**: Find higher-energy moments; ensure mix of clip types
- **If scored low on Content**: Write better titles; add clearer context for each clip
- **If scored low on Platform**: Specify trim points for TikTok; add caption suggestions
- **If scored low on Discoverability**: Choose better thumbnail frames; add trending hashtags

## Update Rules

- Keep 5-10 clips total; remove lowest-energy ones if adding new
- Every clip needs 3 title options (clickbait, descriptive, humorous)
- Timestamps must be exact: "1:23:45 - 1:24:12" not "around 1:23"
- Platform recommendations must be specific: "Trim to 15s for TikTok, start at [timestamp]"
- Thumbnail suggestions should note the specific facial expression or action
- Include at least one clip from each major type if present in stream

## Quality Check

Before resubmitting:
- Would you click on these clips if you saw them in your feed?
- Are timestamps precise enough to execute without re-watching?
- Do titles accurately represent the moment while still being engaging?
- Is platform fit clearly explained with actionable guidance?
- Are there 3-5 truly viral-worthy moments, not just "okay" ones?
